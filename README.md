# DiamondKata

The outline of this exercise is as follows:

Given 'a', print "a"

Given 'b', print the following:

    "_a_"
    "b_b"
    "_a_"

Given 'c', print the following:

    "__a__"
    "_b_b_"
    "c___c"
    "_b_b_"
    "__a__"

and so on...

## Installation

Clone the repo

run:

```
rake install
```

run:

```
diamond_kata
```

in the command line to start
